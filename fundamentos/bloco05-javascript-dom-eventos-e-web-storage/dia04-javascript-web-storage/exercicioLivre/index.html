Trabalhar em equipe é saber lutar em conjunto por um mesmo objetivo. Nem sempre é fácil manter a união e a concordância entre todos quando uma equipe é composta por diferentes individualidades. Mas é importante saber separar o que é individual do que é de interesse para o grupo. A vitória apenas será alcançada se todos se unirem e trabalharem em conjunto. No final todo sacrifício valerá a pena e sei que pequenos desentendimentos serão esquecidos.
